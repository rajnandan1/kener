Welcome to **Kener** - the modern, open-source status page system. This documentation will help you understand how to set up, configure, and use Kener effectively.

## What is Kener?

Kener is a powerful status page application that helps you:

- **Monitor** your services in real-time
- **Track** uptime and performance metrics
- **Communicate** incidents to your users
- **Customize** the look and feel to match your brand

## Key Features

### Real-time Monitoring

Kener supports multiple monitor types to keep track of your services:

- **API Monitors** - Check HTTP/HTTPS endpoints
- **Ping Monitors** - Verify server availability
- **TCP Monitors** - Test port connectivity
- **DNS Monitors** - Validate DNS records
- **SSL Monitors** - Track certificate expiration

### Incident Management

Keep your users informed with:

- Incident creation and updates
- Scheduled maintenance windows
- Historical incident tracking
- Email notifications

### Customization

Make Kener your own:

- Custom branding and themes
- Multiple language support
- Embeddable widgets
- API access for integrations
